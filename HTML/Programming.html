Programming

Project Scope

Create a mini Library Management System which the user can access via a CLI (Command-Line Interface) using Python. The system must be capable of carrying out CRUD (Create, Read, Update, Delete) processes.

Role and Solution

As the sole developer of this project, my role was to decide how to code the features of the library management system in a way which is user-friendly.


The idea which I came out with was to create a menu, which the user is exposed to as soon as he/she opens the programme. Validation of outputs were also considered in the design process.

Work Process

Here is the general flow used in developing the mini Library Management System:


  1. Determine the purpose of features
  2. Decide on features' functionalities
  3. Consider the flow of features used by users
  4. Development of system using Python
